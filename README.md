## Usage

```
If you clone this repo for test -
add .env file and inside add NODE_ENV,  add your MONGODB_URI and add your PORT
```

```
 npm install
 cd client npm install
 cd ..

 # Run front and backend
 npm run dev

 # Backend only
 npm run server

 # Frontend only
 npm run client

 # Build client
 cd client
 npm run build

 # Prod
 npm start
```
